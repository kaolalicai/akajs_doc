akajs

Kalengo Node 后端 web 项目工具集

快速上手 →

Kalengo 定制

akajs 并不是一个Web框架，它是 Kalengo 的一堆 Node.js 的后端开发实践，其目的是把这些开发实践变成一个包，方便开发快速搭建一个后端服务。

TypeScript 驱动

使用 TypeScript 编写，参考 nestjs egg.js sails.js 等框架的优秀设计，结合 Kalengo 的实际需求，基于 Koa 定制各种工具包，

CRUD

akajs 可以帮你快速生成 CRUD 接口，减少低级重复工作。
